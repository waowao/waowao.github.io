非气泡提示模式示例（默认）

<!-- 通过添加yw-form类使组件自动调用 -->
<form class="yw-form">
    <!-- 使用规则，错误提示默认，展示位置默认在同级yw-error-input类的标签内 -->
    <div>
        <input type="text" class="yw-validator" data-rule="required">
        <span class="yw-error-input"></span>
    </div>

    <!-- 指定错误提示和展示位置，寻找yw-valid-wrap类祖先元素下的yw-error-input展示 -->
    <div class="yw-valid-wrap">
        <div>
            <input type="text" class="yw-validator" data-rule="required" data-msg="请输入内容">
        </div>
        <span class="yw-error-input"></span>
    </div>

    <!-- 多条目验证 -->
    <div class="yw-valid-wrap">
        <input type="text" class="yw-validator" data-rule="required|length:6,16|number"
            data-msg="请填写电话号码|电话号码必须为%d~%d位">
        <span class="yw-error-input"></span>
    </div>

    <!-- 复选组的验证 -->
    <div class="yw-valid-wrap">
        <div class="yw-validator" data-rule="checked:2">
            <input type="checkbox" value="1"/> 标签1
            <input type="checkbox" value="2"/> 标签2
            <input type="checkbox" value="3"/> 标签3
            <input type="checkbox" value="4"/> 标签4
            <input type="checkbox" value="5"/> 标签5
        </div>
        <span class="yw-error-input"></span>
    </div>
</form>

气泡提示模式示例

<form id="demo">
    <!-- 使用规则，错误提示默认，展示位置默认在右方 -->
    <input type="text" class="yw-validator" data-rule="required">

    <!-- 指定错误提示和展示位置（弹出提示模式） -->
    <input type="text" class="yw-validator" data-rule="required" data-msg="请输入内容" data-align="top">

    <!-- 多条目验证 -->
    <input type="text" class="yw-validator" data-rule="required|length:6,16|number"
        data-msg="请填写电话号码|电话号码必须为%d~%d位">

    <!-- 复选组的验证 -->
    <div class="yw-validator" data-rule="checked:2">
        <input type="checkbox" value="1"/> 标签1
        <input type="checkbox" value="2"/> 标签2
        <input type="checkbox" value="3"/> 标签3
        <input type="checkbox" value="4"/> 标签4
        <input type="checkbox" value="5"/> 标签5
    </div>
</form>

    popError: true, // 使用弹出式错误提示
    func: {// 定义验证函数

            return val%2==0;
        }
    }
});

参数

提交整个表单时会自动验证。每个验证的表单元素需要增加类名“yw-validator”。

标签参数名称 默认值 描述 可能的值
data-rule 无验证 验证规则 required，number，length:\d*,\d*，...
data-msg 默认错误提示 遇到错误时的提示信息 任意字符串，%d替代替换字符
data-align right 提示信息展示的位置，仅气泡模式有效 top，right，left，bottom

以下为验证规则列表：

规则名称 语法 描述
required required 此项必须填写，两端空格会被忽略，只有空格也被认为是未填写
number number 必须是数字格式
email email 必须是邮箱格式
length length:(\d*),(\d*) 长度需要在$1和$2之间
max max:(\d*) 最大值为$1
min min:(\d*) 最小值为$1
func func:([_\c\d]+) 使用函数$1对输入验证，通过js参数传入
selected selected 下拉列表必须选中，验证规则需要放在外层元素上
checked checked:(\d+) 单选或复选框必须选中$1个，验证规则需要放在外层元素上

提交整个表单时会自动验证。每个验证的表单元素需要增加类名“yw-validator”。

标签参数名称 默认值 描述 可能的值
popError false 是否采用弹出气泡提示 boolean
parentClass .yw-valid-wrap 表单包裹元素，用于错误展示，不填时寻找的是yw-validator父元素，错误展示在此元素下的errorClass中 任意jQuery可用元素选择器
errorClass .yw-error-input 表单错误展示元素，用于错误展示 任意jQuery可用元素选择器
func {} func过滤条件时用于传入定义的验证方法 js对象，指定方法集
